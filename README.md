# soheilaghayour.github.io

Personal portfolio of Soheila Ghayour Kazemi — Frontend Engineer specializing in
complex, data-heavy interfaces (SQL query builders, KPI dashboards, data
visualization).

Live at <https://soheilaghayour.github.io>.

## Stack

- Vite + React + TypeScript
- React Router (SPA; GitHub Pages deep links via the `404.html` redirect)
- Framer Motion
- Vitest + React Testing Library

## Development

```bash
npm install
npm run dev      # local dev server
npm test         # run tests
npm run lint     # oxlint
npm run build    # production build to dist/
```

Deployed automatically to GitHub Pages by GitHub Actions on every push to
`main`.
